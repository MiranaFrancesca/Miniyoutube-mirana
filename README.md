# Mini YouTube Clone

Application développée individuellement en React.

## Description

Mini clone de YouTube reproduisant ses fonctionnalités principales : navigation, lecture de vidéos, playlist, historique et recherche.

## Fonctionnalités

- Page d’accueil avec 30+ vidéos
- Page de lecture individuelle
- Ajout de vidéos à une playlist
- Historique de visionnage
- Page vidéos verticales (Shorts)
- Barre de recherche par titre
- Connexion simplifiée avec sessionStorage

## Technologies utilisées

- React
- React Router
- Bun
- JSON (données vidéos)
- localStorage
- sessionStorage
- CSS

## Installation

Avec Bun :

```bash
bun install
bun run dev
